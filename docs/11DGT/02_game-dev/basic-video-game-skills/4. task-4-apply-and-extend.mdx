### Kids Can Code Recipes

![Godot Platform Tutorial 2026](/img/11dgt/logo-kids-can-code.png)

Kids can code is an awesome website that has a long list of "recipes" which are stand alone parts which can be incorporate into a variety of games.

- [Audio Manager](https://kidscancode.org/godot_recipes/4.x/audio/audio_manager/index.html) \- useful for adding sound effects for your mob/coin/etc.
- [User Interfaces](https://kidscancode.org/godot_recipes/4.x/ui/index.html)
- [Moving Platforms](https://kidscancode.org/godot_recipes/4.x/2d/moving_platforms/index.html)

An excellent resource for finding colour pallettes is [Lospec Palette List](https://lospec.com/palette-list).
